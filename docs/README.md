# 速派 · 文档

## 阅读顺序

| # | 文件 | 读什么 | 
|:--:|------|--------|
| 1 | `PLAYBOOK.md` | 设计系统 + 工程规范 + Red Flags |
| 2 | `METHODOLOGY.md` | 怎么构建（新项目复用） |
| 3 | `REFERENCES.md` | 外部范式参考（gstack · superpowers · Claude · OpenAI） |
| 4 | `ARCHITECTURE.md` | 技术架构 |

| 6 | `prd/` | 当前 Sprint PRD |

## 规则

- 一个主题一个文件
- 不加日期前缀（用 Git 追踪版本）
- 文件名大写 = 核心文档，小写 = 参考
- 禁止"v1_final_revised.md"这种命名
