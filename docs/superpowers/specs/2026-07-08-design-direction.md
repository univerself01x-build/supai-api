## 参考产品分析

### 1. Linear — 项目卡片列表的黄金标准

```
Linear 的核心设计语言：

  暗色背景 (#0D0E11) + 半透明卡片 (rgba(255,255,255,0.03))
  项目列表 = 紧凑的行，不是卡片墙
  每个项目一行：icon + title + status + assignee + due date
  悬停时浮现操作按钮（不是一直显示）
  状态用颜色点（🟢🟡🔴），不靠文字
  空状态：一句话引导 + CTA 按钮

速派可借鉴：
  ✅ 项目列表 = 紧凑行，不是大卡片
  ✅ 状态可视化（颜色点而非文字标签）
  ✅ 悬停才显示操作
  ✅ 空状态不空白
```

### 2. Vercel/v0 — AI 产出物的展示方式

```
v0 的设计语言：

  对话在左，产出在右（不是上下堆叠）
  AI 生成的内容直接渲染为可交互组件，不是 Markdown 文本
  加载态：骨架屏（shimmer），不是转圈
  每个生成结果可独立操作（复制/编辑/分享）

速派可借鉴：
  ✅ 匹配结果直接渲染为可操作卡片，不是 JSON
  ✅ 加载态用骨架屏（等待匹配时）
  ✅ 每个匹配结果独立可操作
```

### 3. Stripe Dashboard — 数据面板的信息层级

```
Stripe 的设计语言：

  大量留白，信息密度低
  每页只回答一个问题（"这个月收入多少" / "这笔支付什么状态"）
  数字大、标签小、图表克制
  颜色仅用于传达状态（绿=好、红=问题、黄=注意）

速派可借鉴：
  ✅ 运营面板每页只回答一个问题
  ✅ 数字大、标签小
  ✅ 颜色仅用于状态，不用装饰
```

### 4. Raycast — 命令面板的交互模式

```
Raycast 的设计语言：

  ⌘K 打开命令面板 → 打字 → 即时结果
  不需要找按钮、点菜单
  高频操作用快捷键，低频操作用搜索
  AI 功能集成在命令面板里（不是独立聊天窗口）

速派可借鉴：
  ✅ 峰哥/客服高频操作 → 搜索/命令直达（不是翻页找按钮）
  ✅ 客户下单 → 自然语言输入 → 匹配 → 确认（像搜索一样简单）
```

## 速派创意方向

### 核心交互理念：像 Linear 一样克制，像 v0 一样智能

```
不要的：
  ❌ 大卡片墙（占空间，信息密度低）
  ❌ 侧栏导航（当前功能不够多，不需要）
  ❌ 彩色装饰（AI 行业用户反感花哨）
  ❌ 转圈加载（AI Native 用户期待流式反馈）

要的：
  ✅ 紧凑项目行 + 点击展开详情面板
  ✅ 暗色为主，单色强调（Linear 蓝 或 Vercel 紫）
  ✅ 骨架屏加载 + 流式文本
  ✅ 空状态有引导："还没有活动项目。描述你的需求，AI 帮你匹配视觉 Team。"
```

### 交互细节规则

| 规则 | 做法 |
|------|------|
| 状态只用颜色点 | ● 确认 ○ 待匹配 ◉ 进行中 ◎ 已完成 |
| 悬停才出操作 | 默认只读，hover 显示 [确认] [更换] [展开] |
| 输入即搜索 | 顶部输入框同时是搜索 + 新建入口 |
| 骨架屏 > 转圈 | 加载时显示灰色占位块，150ms shimmer |
| 空状态有引导 | 永远不空白，始终有下一步提示 |
| 移动端同等重要 | 手机打开 = 紧凑版，功能不减 |
| 2px 状态条 | iTerm2 风格。输入框顶部 2px 动态色彩条。空闲灰、匹配蓝紫渐变、完成绿闪、需审查琥珀、异常红。不弹窗不转圈 |

### 颜色系统

```
背景:     #0A0A0B（最深） / #111113（卡片） / #1A1A1D（悬停）
文字:     #EDEDEF（主） / #8B8B90（次） / #5C5C60（禁用）
强调:     #2383E2（Linear 蓝，仅用于操作按钮和链接）
状态:     🟢 #30A46C（完成） 🟡 #F5C542（注意） 🔴 #E5484D（问题）
```

## 和竞品的核心差异

| | Calendly 式预约 | Notion 式管理 | 速派 |
|--|---------------|-------------|------|
| 输入 | 选时段 | 写文档 | **说需求 → AI 匹配** |
| 产出 | 日历事件 | 页面 | **Team + 时间线 + 分发追踪** |
| 用户操作 | 手动填 | 手动搭 | **AI 做了，人确认** |

速派不是"又一个 SaaS"，是 **AI 替你组队**。你描述需求，平台返回一支队伍。交互应该反映这个核心差异——输入框是主角，不是表单。

---

## 总结

4 个参考产品的共同点：**克制、暗色、流式、不装饰**。

速派的差异化：**AI 匹配 Team，不是选时段/写文档/填表单。**

Phase 1 前端只需三个元素：搜索/输入框 + 项目行列表 + 展开面板。够用、不花哨、AI Native。
