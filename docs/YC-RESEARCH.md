# YC 垂直 AI Agent 研究

> 分析时间: 2026-07-03
> 基于 YC W24-S26 批次公开数据

## 核心趋势

YC 不再投"通用 AI"公司。2025-2026 年投资集中在三类:

### 1. 垂直 Agent (占比最高)
```
医疗 Agent — AI 医生助手，处理预约/病历/保险
法律 Agent — 合同审查、合规检查
房地产 Agent — 自动找房、评估、协商
金融 Agent — 个人理财 AI 顾问
教育 Agent — AI 辅导老师、课程生成
```

**模式: 通用模型 + 行业知识库 + 工作流自动化**

### 2. Agent 基础设施
```
记忆层 — 长期上下文存储（Mem0, Honcho）
工具层 — 浏览器/终端/API 调用抽象
编排层 — 多 Agent 协调（CrewAI 获投）
评估层 — Agent 行为测试框架
```

### 3. AI 原生产品
```
不是"给旧产品加 AI"，而是从 0 用 AI 构建
例: AI 律师（不是律师事务所用 ChatGPT）
    AI 设计师（不是 Figma 加 AI 插件）
```

## 对我们的启示

| YC 模式 | 速派应用 |
|---------|---------|
| 垂直 > 通用 | 摄影派单 > 通用匹配平台 |
| 工作流 > 聊天 | 匹配→通知→确认→交付 闭环 |
| 行业知识 > 模型能力 | 摄影活动分类 > 更好的 prompt |
| B2B 优先 | 卖给企业（活动公司）> 卖给个人 |
