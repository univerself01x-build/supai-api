# Stanford CS 224G

**Stanford CS 224G 课程** · Stanford CS 224G: Building and Scaling LLM Applications · /ˈstænfəd siː ɛs tuː tuː fɔː dʒiː/

Stanford 唯一直接对标 AI Engineer 角色的课程。2026 Winter 由 John Whaley（CS PhD, 连续创业者）和 Jan Jannink（CS PhD）执教。10 周 4 个 Sprint，最终 Demo Day。

## 课程大纲

| 周 | 主题 | 峰哥相关度 |
|---|------|:---:|
| W1 | LLM 格局 + AI 开发原则 | ⭐⭐ |
| W2 | **Reasoning Models**（o1/o3, DeepSeek R1, Claude Sonnet 4） | ⭐⭐⭐ |
| W3 | **Context Engineering & RAG** | ⭐⭐⭐⭐⭐ |
| W4 | **Agentic Workflows**（ReAct, Multi-Agent, Tool Use） | ⭐⭐⭐⭐⭐ |
| W5 | **Production Systems & Evaluation** | ⭐⭐⭐⭐⭐ |
| W6 | Code Generation & Advanced Agents | ⭐⭐⭐ |
| W7 | RAG, Memory & Orchestration（含 PydanticAI） | ⭐⭐⭐⭐ |
| W8 | Data Strategy & Memory Layer | ⭐⭐⭐⭐ |
| W9 | Realtime Voice AI & Safety | ⭐⭐ |
| W10 | Pitching & Demo Day（Dalton Caldwell 客座） | ⭐⭐ |

## 为什么这对 AI Agent 工程师重要

这是目前 Stanford 对 AI Engineer 最精准的课程——不含一行数学推导，全实战。峰哥不需要上，但重点看 W3（Context Engineering）、W4（Agentic Workflows）、W5（Production Systems）三周的讲义和视频。

## 嘉宾讲师

- Dalton Caldwell（YC / Standard Capital）— AI 创业投资
- Andy Bromberg — 约束理论应用于 LLM 产品
- Leonie Freisinger — PydanticAI + 制造业 AI
