# AI Engineer

**AI 工程师** · AI Engineer · /eɪ aɪ ˌɛndʒɪˈnɪə/

用 AI 模型（API）构建产品的工程师。不训练模型，而是编排模型能力。由 swyx（Latent Space）在 2023 年奠基性文章《The Rise of the AI Engineer》中定义。

## 与 ML Engineer 的区别

| | AI Engineer | ML Engineer |
|---|------------|-------------|
| 工作 | 用模型 API 构建产品 | 训练/微调模型 |
| 核心技能 | 软件工程 + API 编排 | 数学 + PyTorch + 分布式训练 |
| 需要 PhD？ | 不需要（Karpathy 原话） | 通常需要研究背景 |
| 数量预测 | 5 年内超过 ML Engineer | 当前数量更多 |

## 为什么这对 AI Agent 工程师重要

峰哥的定位是 AI Engineer 而非 ML Researcher。这意味着：
- ❌ 不需要学 PyTorch 底层、手算梯度、分布式训练
- ✅ 需要掌握 Agent 编排、Context Engineering、Evals 体系
- ✅ 核心竞争力是"把 AI 能力变成产品"
